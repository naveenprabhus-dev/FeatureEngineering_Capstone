# Hotel Booking Cancellation Prediction

This project focuses on predicting hotel booking cancellations using machine learning.

## Tasks Covered
- Baseline Model
- Curse of Dimensionality
- Numeric Preprocessing
- Distance Metrics (KNN)
- Pipeline (with ColumnTransformer)
- Feature Extraction
- Feature Construction
- Feature Importance & Selection

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Results
Feature engineering and preprocessing significantly improved model performance.
